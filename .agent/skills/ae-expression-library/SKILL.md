# After Effects エクスプレッション辞書

After Effects のエクスプレッションをカテゴリ別に整理した包括的なリファレンス辞書。
ユーザーからエクスプレッションに関する質問があった場合、このスキルの辞書ファイルを参照して正確な情報を提供する。

---

## 🔴 使用ルール

> [!IMPORTANT]
> - ユーザーの質問に最も関連する **カテゴリファイルを参照** してから回答すること
> - エクスプレッションのコードは **辞書の記載内容を基準** とし、独自の推測で書かないこと
> - 複数カテゴリにまたがる質問の場合は、関連する **複数のファイルを参照** すること

> [!CAUTION]
> **作例保存ルール**: 会話の中でユーザーと一緒に作成したエクスプレッション作例は、**必ず `gallery/` フォルダに保存** すること。以下のルールに従う：
> 1. **カテゴリ別ファイル** に分類する（例: `grid_examples.md`, `text_examples.md`, `audio_examples.md`）
> 2. 既存カテゴリに該当するものは **既存ファイルに追記**、新しいカテゴリなら **新ファイル作成**
> 3. 各作例は以下の統一フォーマットで記述すること：
>    - 概要 / プロパティ数 / レイヤー数 / 難易度
>    - AE ワークフロー（手順）
>    - 各プロパティのエクスプレッション全文（コピペ可能）
> 4. SKILL.md の「作例ギャラリー」セクションにも新ファイルを追記する

| # | 名前 | コード | カテゴリ |

### 用途別索引

#### アニメーションを作りたい
- 揺れ・ブレを加えたい → **06 ウィグル**
- 滑らかに動かしたい → **04 補間・イージング**
- 繰り返し動かしたい → **05 ループ**
- 円形に動かしたい → **01 基本プロパティ** → 円運動
- バウンスさせたい → **01 基本プロパティ** → バウンス/スプリング

#### 見た目を変えたい
- 色を変えたい → **09 カラー**
- フェードイン/アウト → **01 基本プロパティ** → Opacity
- テキストを動的に変えたい → **08 テキスト**
- 音に合わせて動かしたい → **11 オーディオ連動**

#### 制御したい
- 他のレイヤーと連動させたい → **07 レイヤー参照**
- 条件によって変えたい → **12 条件分岐**
- マーカーで切り替えたい → **12 条件分岐** → マーカー
- 3Dで制御したい → **10 3D・カメラ**

#### 効率化したい
- コントローラーで一括制御 → **07 レイヤー参照** → コントローラーパターン
- 複数レイヤーに一括適用 → **13 実践レシピ** → カスケード
- グリッド/円形に自動配置 → **13 実践レシピ** → レイアウト

## 📖 辞書エントリのフォーマット

各辞書ファイルのエントリは以下のフォーマットで統一されています：

```
### 📌 エクスプレッション名
**用途**: 何のために使うか
**適用先**: Position / Scale / Rotation / Opacity / Any
**難易度**: ⭐ / ⭐⭐ / ⭐⭐⭐

（コード例）
（パラメータ表）
（図解）
（TIP / WARNING）
```

### 難易度の目安

| 難易度 | 意味 |
|--------|------|
| ⭐ | 初心者OK。コピペですぐ使える |
| ⭐⭐ | 中級。パラメータのカスタマイズが必要 |
| ⭐⭐⭐ | 上級。仕組みの理解が必要 |
