How It Works

自動振り分けフロー

README/マニュアルの指示を解析し、指示通りにファイルを配置します。
憶測では動かず、根拠が明確です。

📦 ZIP / フォルダ 複数選択可
→
📝 ドキュメント解析 .md .txt .pdf
→
🔍 配置指示を抽出 ファイル名 → パス
→
✅ 確認 & インストール 根拠を表示
💡
重要： ドキュメントに配置指示がないファイルは自動的に対象外となります。
指示があるファイルだけを、指示通りの場所に配置します。
Features

主な特徴

📝

ドキュメント自動解析

README.md / .txt、マニュアル、PDFなど全てのドキュメントを横断的にスキャンして配置指示を抽出します。

📦

ZIP & 複数選択

ZIPファイルを自動展開して処理。⌘クリックで複数フォルダ/ZIPをまとめてバッチインストール。

🔑

管理者権限の自動対応

/Applications/ への書き込みで権限エラーが発生した場合、macOSのパスワードダイアログを自動で表示します。

🎯

根拠の明示

各ファイルの配置先について「READMEのどの行から判断したか」を表示。安心してインストールできます。

🗂️

フォルダごとコピー

「NeoUtility.jsx + NeoUtilityフォルダ」のような、ファイルとフォルダの両方のコピーにも対応。

🖥️

GUI & CLI 両対応

ダブルクリックでGUI起動、ターミナルからCLI実行も可能。スクリプトによる自動化にも対応します。

Setup

セットアップ

事前準備は Python 3 のインストールだけ。すぐに使い始められます。

1

Python 3 を確認

ターミナルで以下を実行して、Python 3 がインストール済みか確認してください。

python3 --version

Python 3.x.x と表示されればOK。なければ Homebrew でインストール：

brew install python3
2

好きな場所に配置

ダウンロードしたフォルダを好きな場所に置いてください。

例: ~/Applications/AE-Installer-Tool/
3

起動する

方法A: AE Installer Tool.app をダブルクリック
方法B: ターミナルで実行

python3 ae_installer.py
⚠️
初回起動時に「開発元不明」と表示される場合：
Finder で .app を右クリック → 「開く」 を選択。
またはシステム設定 → プライバシーとセキュリティ → 「このまま開く」をクリック。
Usage

使い方（GUI）

ダイアログに従って操作するだけ。3ステップでインストール完了。

1

AE バージョンを選択

インストール先の After Effects バージョンを選びます。
Mac にインストール済みの AE が自動で検出されます。

2

フォルダ or ZIP を選択

「📁 フォルダ」か「📦 ZIP」を選択し、インストールしたいプラグイン/スクリプトを選択。
⌘クリックで複数選択して一括処理もOK。

3

確認してインストール

README/マニュアルの指示から検出されたファイルと配置先が一覧表示されます。
内容を確認して「OK」を押すとインストールが実行されます。
権限が必要な場合は macOS のパスワードダイアログが表示されます。

✅
インストール後は After Effects を再起動してください。
スクリプトは「ウィンドウ」メニューから、プラグインは「エフェクト」メニューから利用できます。

CLI Mode

コマンドライン（CLI）

ターミナルから直接実行。自動化スクリプトにも組み込めます。

# フォルダを指定してドライラン python3 ae_installer.py --dry-run /path/to/plugin -v 2026 # ZIPファイルをインストール python3 ae_installer.py /path/to/plugin.zip -v 2026 # 複数パッケージを一括処理 python3 ae_installer.py /folder1 /folder2 /file.zip -v 2025 # バージョン省略（最新版に自動選択） python3 ae_installer.py /path/to/plugin
オプション説明
-v / --versionAE バージョンを指定（例: 2025, 2026）
-d / --dry-runドライラン（実際にはコピーしない）

FAQ

よくある質問

「配置指示が見つかりません」と表示される ▼
README やマニュアルに「Scripts/ScriptUI Panels」「Plug-ins」「Presets」などの AE ディレクトリ名とファイル名がセットで記載されている必要があります。ドキュメントが同梱されていない、または配置先の記載がない場合は手動でインストールしてください。
Windows で使える？ ▼
現在は macOS 専用です。Windows版は今後対応予定です。
なお、.aex（Windows用プラグイン）は自動的にスキップされます。
パスワードを求められるのはなぜ？ ▼
After Effects は /Applications/ にインストールされており、書き込みに管理者権限が必要です。macOS のパスワードダイアログが表示された場合は、Mac のログインパスワードを入力してください。
インストール後にプラグインが見つからない ▼
After Effects を完全に終了して再起動してください。
スクリプトは ウィンドウ メニュー、プラグインは エフェクト メニューに表示されます。
PDFの配置指示も読み取れる？ ▼
はい。PyPDF2 または pdfplumber がインストールされていれば、PDF からもテキストを抽出して配置指示を検出します。
pip3 install PyPDF2 でインストールしてください。
