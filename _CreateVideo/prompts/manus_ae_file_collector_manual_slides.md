# AE File Collector マニュアルスライド作成プロンプト

## 基本設定
- **スライド形式**: 16:9 アスペクト比
- **言語**: 日本語
- **対象**: Adobe After Effects ユーザー（初心者〜中級者）
- **スライド枚数**: 15〜20枚程度

---

## 制作指示

以下の内容で、AE File Collector v3.0 の操作マニュアルスライドを作成してください。

### 全体的なデザイン方針
- **カラースキーム**: ダークグレー（#1a1a2e）+ パープル/ブルー系アクセント（#4a4a8a）をベースに、After Effects のブランドカラーを意識
- **フォント**: モダンで読みやすいサンセリフフォント
- **スタイル**: プロフェッショナルでクリーンなビジネススタイル
- **アイコン**: 各機能を視覚的に表現するアイコンを活用

### スライド 3: AE File Collector とは
```
概要説明:
- After Effects プロジェクトで使用されているすべてのフッテージファイルを
  自動的に収集し、整理された構造でエクスポートするスクリプト
- プロジェクトの引き渡しやバックアップに最適
- 日本語完全対応
```

### スライド 4: 主な機能一覧
```
✅ 収集機能: 使用ファイルを収集して別フォルダにエクスポート

✅ AI/PSD/PDFのレイヤー情報を保持
✅ プリセット保存・読み込み
```

```
2️⃣ After Effects を再起動

3️⃣ メニューから「ウィンドウ → AEFileCollector_App.jsx」を選択
```

### スライド 10: 収集と整理の違い
```
🔵 収集ボタン
━━━━━━━━━━━━━━━━━━━
プロジェクトで使用されているファイルを
外部フォルダにコピーして収集
→ プロジェクトの引き渡し・バックアップに

🔵 整理ボタン
━━━━━━━━━━━━━━━━━━━
プロジェクトパネル内の構造を整理
（ファイルの物理コピーは行わない）

```

### スライド 11: 収集後のフォルダ構造
```
収集先フォルダ/
├── AEP/                  ← プロジェクトファイル
├── Footage/
│   ├── IMAGE/           ← 画像ファイル（AI/PSD/PNG/JPG等）
│   ├── MOVIE/           ← 動画ファイル（MP4/MOV等）
│   ├── AUDIO/
│   │   ├── BGM/        ← BGM
│   │   ├── SE/         ← 効果音
│   │   ├── NARR/       ← ナレーション
│   │   └── _OTHER/     ← その他音声
│   └── 3D/              ← 3Dファイル（C4D/OBJ/EXR等）
├── FONTS/               ← 収集されたフォント
├── _BackUP/             ← プロジェクトバックアップ
└── CollectedFiles_Report.csv
```

### スライド 12: レポート機能①（CSVレポート）
```
【CollectedFiles_Report.csv】

収集されたファイルの詳細情報を記録:
- No（番号）
- FileName（ファイル名）
- Category（カテゴリ）
- OriginalPath（元のパス）
- NewPath（新しいパス）
- FileSize（ファイルサイズ）
- UsedInComps（使用コンポジション）

💡 Shift-JIS形式で日本語Excelに対応
```

### スライド 13: レポート機能②（フォントレポート）
```
【FontUsage_Report.txt】

プロジェクトで使用されているフォント一覧

【FontCollection_Report.txt】

収集成功/失敗したフォントの詳細:
- 収集成功したフォント
- 収集できなかったフォント
  - macOSシステムフォント（SIP保護）
  - Adobe Fonts（DRM保護）
  - 見つからなかったフォント

💡 Shift-JIS形式で文字化けなし
```

### スライド 14: AI/PSD/PDFファイルの取り扱い
```
⚠️ レイヤーファイル（AI/PSD/PDF）について

これらのファイルは【自動リンク更新の対象外】です

理由:
After Effects でレイヤーを個別にインポートした場合、
自動リンク更新でレイヤー情報が失われる可能性があるため

対処法:
収集後、手動でリンクを更新してください
（ファイル → フッテージを置き換え）
```

### スライド 15: トラブルシューティング①
```
よくある質問 Q&A

Q: スクリプトが表示されない
A: After Effects の環境設定で「スクリプトによるファイルへの
   書き込みとネットワークへのアクセスを許可」をオンに

Q: 収集されないファイルがある
A: プロジェクトパネルで対象コンポジションが選択されているか確認

Q: 日本語が文字化けする
A: CSVはShift-JIS形式のためExcelでそのまま開けます
```

### スライド 16: トラブルシューティング②
```
フォント収集について

⚠️ 以下のフォントは収集できません:

1. Adobe Fonts (Typekit)
   → DRM保護のため収集不可

2. macOSシステムフォント（Hiragino等）
   → システム保護(SIP)のため収集不可

3. /System/Library/Fonts/ 内のフォント
   → セキュリティ上の制限

💡 これらはレポートに記録されます
```

## 補足情報

### 対象バージョン
- AE File Collector v3.0
- Adobe After Effects 2024以降を想定

### 主要な用語
- **収集**: ファイルを外部フォルダにコピー
- **整理**: プロジェクトパネル内の構造を整頓
- **リンク更新**: フッテージのパスを新しい場所に更新
- **プリセット**: 設定を保存/読み込み

### 特記事項
- フォント収集機能はデフォルトではOFF（ユーザーによるオプトインが必要）
- リンク更新機能もデフォルトではOFF（AI/PSD/PDFは対象外）
- プロジェクトバックアップは `_BackUP` フォルダに `[プロジェクト名]_[タイムスタンプ]_BK.aep` 形式で保存
